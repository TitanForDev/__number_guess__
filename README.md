# 猜數字遊戲 (Guess the Number) - 沒事找事做

一個現代化、響應式的 1-100 猜數字遊戲，具有動態視覺回饋與慶祝效果。

## 如何執行

1. 安裝相依套件：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 在瀏覽器中開啟顯示的網址（通常是 http://localhost:5173）。

## 遊戲規則

- 系統會隨機產生一個 1 到 100 之間的數字。
- 你有 7 次機會可以猜測。
- 每次猜測後，系統會提示你猜高了還是低了。
- 祝你好運！
